☕ Starbucks Beverage Insights Dashboard

📌 Project Overview

The Starbucks Beverage Insights Dashboard is an interactive Power BI report designed to analyze Starbucks beverage data and uncover key insights related to calories, caffeine, sugar, protein, beverage categories, and global beverage distribution.

This dashboard enables users to explore beverage nutritional information, identify high-caffeine drinks, compare categories, and gain a better understanding of Starbucks' product offerings through visually appealing and interactive reports.

🎯 Objectives
Analyze nutritional content across Starbucks beverages.
Identify beverages with the highest caffeine levels.
Compare calorie distribution among beverage categories.
Visualize category-wise beverage distribution.
Enable dynamic filtering using slicers for deeper analysis.

📊 Dashboard Features
KPI Cards

The dashboard includes key performance indicators such as:

Total Beverages: 30
Average Sugar: 31 g
Average Calories: 175 kcal
Average Caffeine: 81 mg
Interactive Filters

Users can filter data using:

Protein Range Slider
Beverage Preparation Type Dropdown

These filters dynamically update all visuals on the report page.

Visualizations
1. Average Calories by Beverage Category
Displays average calorie content across beverage categories.
Helps identify high-calorie and low-calorie beverage groups.
2. Starbucks Global Presence
Shows beverage count distribution across categories.
Highlights the most common beverage categories.
3. Beverage Category Distribution
Donut chart displaying percentage share of each beverage category.
4. Average Caffeine by Category
Bar chart comparing average caffeine content among beverage categories.
5. Top 5 Highest Caffeine Beverages
Image-based visual displaying:
Beverage image
Beverage name
Average caffeine value

Provides a visually engaging representation of the most caffeinated Starbucks beverages.

📈 Key Insights

Frappuccino Blended beverages contain the highest average calorie content.
Highest Caffeine Beverage
